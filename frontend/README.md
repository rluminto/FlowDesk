# FlowDesk Frontend

React, TypeScript, Vite, Tailwind CSS, and React Router frontend for FlowDesk.

## Local Setup

```powershell
npm install
npm run dev
```

Frontend URL:

```text
http://127.0.0.1:5173/
```

## Environment

Create a local `.env` file from `.env.example`:

```text
VITE_API_BASE_URL=http://127.0.0.1:8000/api
```

## Scripts

```powershell
npm run dev
npm run build
npm run lint
```
